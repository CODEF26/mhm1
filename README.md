# مجلة أسرة المحيميد الإلكترونية

مجلة عائلية إلكترونية بمناسبة عيد الفطر المبارك، تحتوي على إنجازات وأخبار أبناء الأسرة.

## المميزات
- تصميم أنيق وعائلي
- متوافق مع الجوال والتابلت
- صور بمعالجة احترافية
- كروت تفاعلية أنيقة

## التقنيات المستخدمة
- HTML5, CSS3
- Bootstrap 5 RTL
- Font Awesome 6
- Google Fonts (Cairo, Tajawal)
- SweetAlert2

## النشر
المجلة متوافقة مع GitHub Pages و Google Apps Script.
